Variables en JavaScript

Una variable refiere, en una primer instancia, a cosas que son susceptibles de ser modificadas (de variar), de cambiar en función de algún motivo determinado o indeterminado

Para declarar variables se utiliza:
Palabras reservadas Descripción
let variable local o de bloque
var valores globales generales
const valores constantes
Ejemplo de variables de JavaScript: 
        
            let nombre='Juan';
            document.write(nombre);
            let nombre=10;
            document.write(edad);
Para poder incluir un script en html se utiliza: 
        
            <script src="codigo.js">
            document.write('Hola');
            </script>
